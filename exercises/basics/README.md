# Django Basics Exercises

This directory contains hands-on exercises to practice Django's core concepts covered in the architecture tutorial.

## Exercise Structure

Each exercise includes:
- **Problem description**: What you need to build
- **Starter code**: Basic files to get you started
- **Solution**: Complete working solution
- **Common mistakes**: Things to watch out for

## Exercises

1. **[Exercise 1: Your First Model](./exercise-1-models/)**
   - Create a Book model with various field types
   - Practice model methods and properties
   - Learn about string representations

2. **[Exercise 2: Simple Views](./exercise-2-views/)**
   - Create function-based views
   - Handle URL parameters
   - Pass context to templates

3. **[Exercise 3: Template Basics](./exercise-3-templates/)**
   - Create templates with Django template language
   - Use template inheritance
   - Apply filters and tags

4. **[Exercise 4: URL Configuration](./exercise-4-urls/)**
   - Set up URL patterns
   - Use named URLs
   - Handle different parameter types

5. **[Exercise 5: Putting It All Together](./exercise-5-integration/)**
   - Build a complete mini-application
   - Connect models, views, templates, and URLs
   - Add basic styling

## How to Use These Exercises

1. Read the problem description in each exercise folder
2. Try to implement the solution yourself first
3. Check your work against the provided solution
4. Review common mistakes to avoid similar issues

## Prerequisites

Before starting these exercises, make sure you have:
- Completed the Django setup tutorial
- Read the Django architecture tutorial
- A working Django development environment

## Getting Help

If you get stuck:
1. Check the troubleshooting section in each exercise
2. Review the architecture tutorial for concept clarification
3. Look at the common mistakes section
4. Try the solution and understand how it works
